# Banco de Dados – Histomap

O banco de dados do sistema Histomap foi modelado em PostgreSQL
e implementado utilizando a plataforma Supabase.

A estrutura foi baseada em um modelo relacional, contemplando
usuários, cidades históricas, roteiros turísticos, avaliações
e conteúdos históricos.

As tabelas foram organizadas de forma normalizada, utilizando
chaves primárias e estrangeiras para garantir integridade
referencial e escalabilidade do sistema.
